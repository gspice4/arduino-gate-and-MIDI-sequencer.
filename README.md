
# arduino-gate-and-MIDI-sequencer.
Made by gspice4 on 4.20.26
This is an advanced-programming type gate and MIDI sequencer that features the Arduino MEGA 2560 Pro.
*Flowchart contains initial signal path of the code and hardware that is needed to run the program.*
Made in the official Arduino IDE
Uses libraries: SPI.h, the Adafruit SSD_1306 and GFX Libraries, SD.h (A work in progress), and MIDI.h (Not currently applied in the code.)
